# Comparative-Fine-Tuning-of-a-Compact-LLM
This study investigates the resource-efficient fine-tuning of a compact 0.6-billion parameter model (Qwen3-0.6B) for expert-level biomedical QA. Using the PubMedQA dataset, Full Fine-Tuning (FFT) was compared against advanced adaptation techniques, including QLoRA, Knowledge Distillation (KD) and Direct Preference Optimization (DPO).
